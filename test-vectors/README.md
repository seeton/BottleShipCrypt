# Test vectors

This directory is reserved for future deterministic BottleShip fixtures.

Planned contents include vectors for both:

- weak-demo behavior
- strong-model simulator / proof-target behavior

Examples:

- seal and decrypt vectors
- threshold refusal cases
- destroyed capsule cases
- corrupted manifest cases
- rollback rejection cases against versioned authenticated roots

The current repository ships only the weak browser demo and strong-model documentation, so no formal vectors are included yet.
