⚠ Weak-model warning
Unsafe for real secrets: this is a weak-model visualization only. The browser cannot enforce irreversible capsule destruction, rollback-resistant state, trusted key custody, or copy resistance. A user can inspect or modify the JavaScript, duplicate archive state, or replay an earlier snapshot.

Use this page to understand the BottleShip flow only. For the current non-browser simulator path, use the Go CLI in cmd/bship. The prototype/ directory remains reserved for future prototype notes and stronger-model work.

Weak browser demo

  • Runs entirely in user-controlled JavaScript.
  • Simulates chunk capsules only in browser memory.
  • Can illustrate threshold refusal, keep-set selection, and prune flow.
  • Cannot stop copying, rollback, or code tampering.

Strong trusted-state model

  • Needs a trusted component to hold capsule and state custody.
  • Requires rollback-resistant state across operations.
  • Must irreversibly destroy capsules when pruning.
  • Is the only path for meaningful security claims.
